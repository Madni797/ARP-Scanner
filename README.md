# ARP Scanner Using Python

## Objective

The objective of this project is to develop an ARP Scanner using Python and the Scapy library to discover active devices on a local area network by displaying their IP and MAC addresses.

## Tools Used

- Python 3
- Scapy
- Visual Studio Code
- Windows
- PowerShell

## Commands Used

```bash
python -m pip install scapy --break-system-packages
ipconfig
python arp_scanner.py
```

## Output

The program scans the specified IP range and attempts to display the IP and MAC addresses of active devices on the local network.

## Skills Learned

- Python Programming
- ARP Protocol
- Scapy Library
- Network Discovery
- Basic Network Scanning
